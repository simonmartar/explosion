# Book Library Manager

Organize your personal book collection without using a database.

The application stores books in memory, allows filtering by author, genre,
publication year and reading status, and prints a simple catalog.

---

### Demonstration

```
Library

The Hobbit
J.R.R. Tolkien
Fantasy
1937
Read

-------------------------

Books: 5

Fantasy: 2
Science Fiction: 2
History: 1
```

---

### Included modules

- Book model
- Library storage
- Catalog browser
- Filtering engine
- Statistics report

---

### Build

```
./gradlew run
```

Kotlin/JVM console application.
